Проекты

Generative ML VAX signals

Генеративная сеть ВАХ-сигналов

Проект про генеративную модель для вольт-амперных характеристик: сеть учится структуре экспериментальных кривых и создает похожие сигналы, которые можно использовать для расширения датасета, проверки пайплайнов анализа и поиска устойчивых закономерностей в шумных измерениях.

Важная часть идеи — соединить ML и предметную физико-химическую интуицию: не просто получить красивую кривую, а сохранить форму сигнала, характерные участки, выбросы, шум и масштаб так, чтобы синтетика оставалась полезной для дальнейших моделей.
